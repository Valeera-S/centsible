# Centsible

Local-first expense tracker. Your money data never leaves your device.

Centsible is a PWA that runs in your browser on desktop and phone, stores everything
in IndexedDB on the device, and never talks to a server. Track spending against a
monthly budget, see where the money goes by category, and record an expense in
seconds with one-line natural-language entry, voice input, memo paste, or bank CSV
import.

Status: under active development, not yet released.

## Development

```sh
npm install
npm run dev        # start dev server
npm test           # run unit tests
npm run lint       # eslint
npm run typecheck  # tsc
npm run build      # production build
```
